Blog / Articles

Sharing my journey from a simple HTML file to a full website.

My journey from a simple HTML file to a full-fledged website has been a transformative experience in learning and creativity.

It began with just a blank page and basic HTML tags like `<h1>` and `<p>`, but as my curiosity grew, I started exploring CSS to style my content and make it visually appealing.

Gradually, I learned JavaScript to add interactivity, followed by responsive design techniques to ensure my site looked good on all devices.

I experimented with animations, navigation bars, contact forms, and later integrated tools like GitHub, Firebase, and Netlify to bring my project to life.

Each step came with its own challenges—broken layouts, bugs, and design hurdles—but overcoming them taught me valuable lessons about problem-solving and user experience.

Looking back, it's incredible to see how a single HTML file evolved into a dynamic, hosted website that reflects both my skills and growth as a developer.
